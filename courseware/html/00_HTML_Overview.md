# HTML Courseware — Full Stack Java Developer Track

## Course Overview

This courseware is designed for Java Full Stack Developers who need a thorough, enterprise-level understanding of HTML. You will learn not just how to write HTML, but *why* certain patterns exist, how browsers parse and render markup, and how HTML integrates into production Java web applications using Spring Boot, Thymeleaf, JSP, and modern SPA architectures.

## Learning Outcomes

By the end of this HTML module, you will be able to:

- Write fully valid, semantically correct HTML5 documents from memory
- Understand the browser's critical rendering path and how HTML triggers it
- Build accessible, WCAG 2.1 AA-compliant markup
- Integrate HTML templates into Spring Boot applications using Thymeleaf
- Optimise page load performance using resource hints and lazy loading
- Apply SEO best practices and Open Graph metadata
- Debug HTML structure using browser DevTools
- Understand how Java-generated HTML differs from static files

## Prerequisites

- Basic familiarity with the concept of web pages
- Java programming (Spring Boot is used in integration examples)
