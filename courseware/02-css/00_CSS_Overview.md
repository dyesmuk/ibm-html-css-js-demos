# CSS Courseware — Full Stack Java Developer Track

## Course Overview

This courseware teaches CSS from fundamentals to enterprise-grade patterns. You will learn not just syntax, but how the cascade, specificity, inheritance, and the box model actually work — giving you the mental models to debug any layout problem.

## Learning Outcomes

- Write CSS with confidence, knowing exactly why a rule applies or doesn't
- Understand specificity, inheritance, and the cascade so you can debug style conflicts
- Build fully responsive layouts using Flexbox and Grid
- Apply a professional CSS architecture (custom properties, BEM) to a real project
- Style accessible forms and links
- Integrate CSS effectively into a Spring Boot / Thymeleaf application
